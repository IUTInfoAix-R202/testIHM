## Le jeu Quantik

Quantik est un jeu de stratégie abstrait édité par Gigamic (Nouredine Hilal, 2019). Deux joueurs
posent à tour de rôle leurs pièces sur un plateau 4x4 découpé en quatre zones de 2x2. Une seule règle
de pose (interdit de répéter une forme sur une ligne, une colonne ou une zone, peu importe le joueur)
et une seule condition de victoire (compléter un alignement avec les quatre formes différentes) en
font un sujet idéal pour évaluer la logique (R2.03) comme l'interface (R2.02).

![Apercu de l'application Quantik](src/main/resources/assets/quantik_screenshot.png)

Ce dépôt contient une **proposition de correction** des deux sujets : la logique du jeu dans le
paquet `fr.univ_amu.iut.modele` (sujet R2.03) et l'IHM JavaFX dans le paquet `fr.univ_amu.iut` (sujet
R2.02).
