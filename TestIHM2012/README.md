# TestIHM2012

Sujet du test d'IHM de **mai 2012** - département informatique de l'IUT d'Aix-Marseille, DUT 1ère année.

- **Thème** : Démineur (Minesweeper) en Swing.
- **Statut** : sujet **non retrouvé** lors de l'archivage 2026.

Le sujet original (PDF + sources Java Swing) n'a pas pu être récupéré, ni dans les sauvegardes locales, ni dans les archives de l'organisation [`IUTInfoAix-M2105`](https://github.com/IUTInfoAix-M2105). Il est probablement perdu.

Ce dossier existe pour conserver la trace de l'existence d'une édition 2012, qui était chronologiquement la première occurrence du test terminal d'IHM dans ce module.
